🏋️ Workout Plan App

A simple and powerful workout planning application built with React, TypeScript, and Tailwind CSS.

Create structured training programs, organize exercises by workout type, and follow clear rules for reps, tempo, and rest days.

🚀 Features

📅 Structured workout programs (Push / Pull / Legs or custom splits)

🔁 Exercise tracking (sets & reps)

⏱ Tempo guidance (e.g. 2-2-2 controlled reps)

📱 Clean and responsive UI

⚡ Built with modern React + TypeScript

🎨 Styled using Tailwind CSS

🛠 Tech Stack

React

TypeScript

Tailwind CSS
